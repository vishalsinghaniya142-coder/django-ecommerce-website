# 🛒 MyStore - Full Stack Django E-Commerce Website

A modern and responsive E-Commerce Website built using **Django, HTML, CSS, JavaScript, and SQLite**.

This project allows users to browse products, search products, add items to cart, place orders, and manage products through the Django Admin Panel.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/django-ecommerce-website.git
```

Move into the project

```bash
cd django-ecommerce-website
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the server

```bash
python manage.py runserver
```

Open in browser

```
http://127.0.0.1:8000/
```
